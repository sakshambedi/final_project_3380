How to RUN:


## Windows:
Open the folder dist and run the main.exe
Voila thats it ! 


## Linux:
In the main directory, run make to install all the pre-requirements
Then using shell of your choice run python main.py install

Warning : Tkinker shows an error about about the font in linux because of some issue in the Tkinker Library. Please ignore that can not fixed

Well Done :)
